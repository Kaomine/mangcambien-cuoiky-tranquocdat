\## Hướng dẫn chạy thử nghiệm 

Dự án này đã được biên dịch sang WebAssembly để test trực tiếp bằng camera máy tính.

1\. Mở thư mục `browser`.

2\. Mở file `index.html` bằng trình duyệt web (Khuyên dùng tính năng Live Server trên Visual Studio Code).

3\. Cấp quyền truy cập Camera và đưa hình ảnh các loại hoa vào để kiểm tra.
